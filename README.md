# 🧊 3D Educational FEA Demo

A lightweight, browser-based **Finite Element Analysis (FEA) visualization** built with **Three.js**.
This project demonstrates how material properties like **Young’s Modulus (E)** and **Poisson’s Ratio (ν)** affect deformation in a simple 3D object.

---

## 🚀 Features

* 🌐 Runs entirely in the browser (no installation required)
* 🧠 Interactive material parameters:

  * Young’s Modulus (E)
  * Poisson’s Ratio (ν)
* 🎥 Real-time 3D visualization using Three.js
* 🔄 Animated mesh for better spatial understanding
* ⚡ Instant deformation simulation with one click

## 🧠 Physics Behind It

This demo uses simplified linear elasticity:

* σ = E · ε
* εₗₐₜₑᵣₐₗ = −ν · εₐₓᵢₐₗ

Where:

* **E** = Young’s Modulus
* **ν** = Poisson’s Ratio
